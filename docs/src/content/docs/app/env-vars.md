---

-   `TIME_ZONE`: Set time zone used in Temporal API - default is `Europe/Warsaw`
-   `CITY_AGE`: Override `cityAge` variable
-   `CITY_OFFSET` Override `cityOffset` variable
-   `CITY_ID`: Change city by providing id - default is `lublin`
-   `BASE_URL`: Override base URL
-   `SERVER_PORT`: Specify server port - default is `8069`
-   `SERVER_STANDALONE`: Change sever to standalone mode - default is `false`
